# Thalidomide — A Tragedy and a Redemption

**Discovery Story**  
Marketed as a safe sedative in the 1950s → caused birth defects → later repurposed for cancer.

**Medicinal Chemistry Principle**  
- Chirality: one enantiomer sedative, the other teratogenic.  
- Lesson: enantiomers can interconvert in vivo.  

**Interactive Idea (future)**  
Switch R ↔ S forms → see different biological outcomes.
